# Delivery API Service

## Running the app

```bash
make all
make run
```

Add the features in the cache service:
```bash
make seed-redis
```

## Running the app in develop mode

```bash
make all
make run-dev
```

Add the features in the cache service:
```bash
make seed-redis
```

## Executing unit tests

And execute the unit tests:
```bash
make unit-test
```

## Executing lint and type checking

And execute the unit tests:
```bash
make type-check
make lint
```
